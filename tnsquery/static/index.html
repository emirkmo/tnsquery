TNSQUERY API

Open source python API and a cached performant database for accessing basic transient data (name, coordinates, type, redshift, etc.) from the Transient Name Server (TNS), The IAU approved repository of astronomical transient data.

Fast programmatic access to transient data from TNS:   Try Me!

No Scientific Credentials Required!

"Science knows no country, because knowledge belongs to humanity..."
- Loius Pasteur

TNSQuery was created to enable open science with a goal of allowing everyone to programmatically access the TNS API, which is locked behind science credentials to prevent abuse. The goal is to present a minimal format and avoid overloading the TNS with queries, while enabling dynamic look-up of transient data in scripts and services, essentially acting as a cache layer for the TNS for API queries which only need basic transient data (mainly coordinates).

To that end, TNSQuery is supported by FLOWS research group at Aarhus University, with the aim of being integrated into the popular open source community package Astropy. (Specifically enabling transient coordinate queries via SkyCoord and basic data via Astroquery). Additionally, leveraring Astroquery, TNSQuery attaches basic cross-match data currently in the form of extinction from SFD (2011) Maps provided by IRSA.
